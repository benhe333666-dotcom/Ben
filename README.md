# 人工智能热点时间线

一个面向中国境内访问的人工智能热点新闻网站：只采集可直接访问的中文公开新闻源，并在页面中统一使用中文标题、中文摘要、中文来源名和中文界面文案。默认每小时更新一次数据。

## 本地运行

```bash
npm install
npm run update
npm run dev
```

## 构建

```bash
npm run deploy:check
```

该命令会先抓取中文新闻源并生成 `public/news.json`，再构建静态网页到 `dist`。

## 数据源策略

当前只使用中国境内通常可访问的中文公开源：

- 量子位
- 雷峰网
- 三十六氪
- 科技之家
- 开源中国
- 博客园新闻

已移除境内访问不稳定或不可访问的来源，例如谷歌新闻、海外官方博客、海外英文媒体和论文站点。新闻标题与摘要会做中文化处理，常见英文术语、产品名和技术缩写会转成中文展示。

## 更新机制

- GitHub Actions：保留每小时构建，用于同步仓库和备份发布产物。
- 本机公网入口：建议由本机 Caddy 直接提供 `dist` 静态目录，避免公网访问依赖境外托管。

## 域名

目标域名：

```text
news.ben1067190.top
```

如果面向中国境内用户，建议在阿里云 DNS 中将该子域名解析到本机公网地址，再由 Caddy 提供静态网页。
