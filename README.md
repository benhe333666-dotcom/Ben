# 人工智能热点时间线

一个面向中国境内访问的人工智能热点新闻网站：保留中文公开新闻源，同时新增国外权威人工智能来源。后台抓取后统一做去重、筛选和中文化转写，页面只展示中文标题、中文摘要、中文来源名和中文界面文案。默认每半小时尝试更新一次数据，以降低定时任务偶发延迟或丢失造成的停更风险。
数据默认只保留最近一周；每次自动更新都会重写新闻数据文件，自动移除一周以前的信息，避免长期运行后存储容量失控。

## 本地运行

```bash
npm install
npm run update
npm run dev
```

## 构建

```bash
npm run deploy:check
```

该命令会先抓取新闻源并生成 `public/news.json`，再构建静态网页到 `dist`。

## 数据源策略

当前保留的中文公开源：

- 量子位
- 雷峰网
- 三十六氪
- 科技之家
- 开源中国
- 博客园新闻

新增的国外权威来源会由后台抓取，不要求访问者浏览器直接连接外站：

- 开放人工智能公司
- 谷歌人工智能博客
- 谷歌研究院
- 代码托管平台官方博客
- 代码托管平台更新日志
- 英伟达官方博客
- 微软人工智能博客
- 模型社区博客
- 论文预印本：人工智能
- 论文预印本：机器学习

新闻标题与摘要会做中文化处理，常见英文术语、产品名、技术缩写和英文单位会转成中文展示。页面可见内容保持中文；原文链接只作为“打开原文”的跳转目标，可能受访问者网络环境影响。

## 更新机制

- GitHub Actions：每小时 17、47 分抓取完整来源、去重并生成中文新闻数据，同时把 `public/news.json` 和 `public/sources.json` 同步回仓库。
- 腾讯云轻量应用服务器：每小时 20、50 分先拉取 GitHub 最新数据，再构建静态网页并由 Caddy 提供 `dist` 目录。

腾讯云默认不直接抓取国外来源，因为部分国外站点在中国境内服务器上不可稳定访问；如需让服务器本机尝试抓取，可在服务环境中设置 `RUN_SERVER_UPDATE=1`。
